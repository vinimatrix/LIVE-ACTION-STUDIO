## 5. Consistency Maintenance Rules

Rules to ensure visual and narrative continuity across episodes.

### Character Consistency Rules
```
1. REFERENCE_REUSE: Always use existing character reference images when available
   - Never generate a new full-body reference if one exists in library
   - For new poses/expressions, generate using existing face/body as base

2. NINJA_GEAR_CONSISTENCY: Core ninja equipment must remain consistent
   - Forehead protector always worn (style may vary by character preference)
   - Weapon pouches, standard ninja tools (kunai, shuriken, wire) consistently present
   - Sandals/ninja footwear standard unless specific story reason (special mission, injury, etc.)

3. CHAKRA/VISUAL_EVOLUTION_TRACKING: 
   - Chakra control and visibility must reflect skill level and training
   - New jutsu/dojutsu manifestations require clear in-story justification (training, power-up, etc.)
   - Dojutsu evolution (Byakugan stages, Sharingan stages, Jougan development, etc.) must follow established patterns
   - Power-up forms (Sage Mode, Bijuu Chakra, Karma stages, etc.) have strict visual progression rules

4. PERSONALITY_CONSISTENCY THROUGH GROWTH: 
   - Core personality traits remain but can mature/develop through experiences
   - Trauma responses must be consistent with established psychology
   - Relationship dynamics evolve but core feelings (loyalty, rivalry, affection) remain traceable

5. HONORIFICS AND SPEECH PATTERNS:
   - Character-specific speech patterns must be maintained (Naruto's "dattebayo", Sasuke's terse speech, etc.)
   - Honorific usage (-san, -kun, -sensei, etc.) must reflect relationship and context accurately
   - Shifts in formality must have clear situational justification (emergency, extreme respect, etc.)

6. SPECIAL MARKINGS TRACKING:
   - Scar accumulation must be logical and permanent unless healed by special means
   - Seal/Karma progression must be tracked and visible when active
   - Temporary markings (battle paint, etc.) must be clearly temporary
```

### Environment Consistency Rules
```
1. LOCATION_CANON: Each named location has established visual canon
   - Konoha maintains consistent architectural layout and key landmarks (Hokage Monument, etc.)
   - Training grounds have consistent equipment placement and environmental wear
   - Recurring battlefields show cumulative damage from previous conflicts (when relevant to plot)

2. ENVIRONMENTAL_WEATHER_LIGHTING_CONSISTENCY:
   - Weather changes must follow logical patterns or have clear supernatural/jutsu explanation
   - Time of day progression must be continuous across linked scenes
   - Lighting sources (sun, moon, permanent installations) must maintain consistent positions/patterns

3. DIMENSIONAL_SPACE_CONSISTENCY:
   - Alternate dimensions/Otsutsuki spaces must maintain consistent internal logic
   - Entry/exit points to/from dimensions should be consistent when reused
   - Environmental hazards in special spaces must follow established rules

4. PROP AND WEAPON CONSISTENCY:
   - Character-specific weapons must maintain design (unless reforged/replaced with story reason)
   - Consumable items (exploding tags, medicine pills) must be replenished logically
   - Summoning contracts must be consistent - same summoning animals unless new contract formed
```

### Jutsu and Ability Consistency Rules
```
1. TECHNIQUE_VISUAL_CONSISTENCY:
   - Same jutsu must look substantially the same each time used (variations allowed for power level)
   - Hand signs for specific jutru must be accurate to established sequences
   - Chakra color must match elemental nature (Red=Fire, Yellow=Lightning, etc.) unless special circumstance
   - Advanced/Kekkei Genkai techniques must maintain distinctive visual signatures

2. CHAKRA_CONTROLLABILITY_PROGRESSION:
   - Genin-level: visible chakra only during significant effort or uncontrolled release
   - Chunin/Jounin-level: ability to suppress visible chakra unless intentionally manifesting
   - Kage/Sannin-level: fine control, can make chakra nearly invisible or shape it precisely
   - Senjutsu/Six Paths level: chakra has distinctive sage or divine quality

3. DOJUTSU_USAGE_CONSISTENCY:
   - Byakugan: veins visible around eyes, near 360° vision (except minor blind spot)
   - Sharingan: tomoe development predictable, abilities tied to tomoe count
   - Rinnegan: specific pattern, abilities consistent with wielder's skill
   - Jougan: progressive pattern revelation, abilities tied to user's understanding/control
   - All dojutsu: activation/deactivation has consistent visual timing and effect

4. SENJUTSU AND SPECIAL ENERGY CONSISTENCY:
   - Sage Mode: distinctive pigmentation around eyes, potential animalistic features
   - Tailed Beast Chakra: distinctive color and appearance matching bijuu
   - Six Paths Chakra: distinctive rinnegan-like patterns in eyes, truth-seeking orbs when applicable
   - Kurama Chakra Mode: distinctive chakra cloak shape and color
```
